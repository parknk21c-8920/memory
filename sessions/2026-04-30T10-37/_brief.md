# 📋 작업 브리프

**원 명령:** 썸네일 3개 만들어줘. 작업이 완료되면 바로 보고해

## 요약
사용자가 요청한 3개의 썸네일 디자인 제작 및 결과 보고.

## 분배
- **🎨 Designer**: 사용자가 요청한 3개의 썸네일 디자인 초안 제작. 디자인 컨셉, 색상, 폰트, 텍스트 등을 포함. 썸네일 제작 후 Google Drive에 업로드.
- **📱 Secretary**: 디자이너가 제작한 3개의 썸네일 디자인을 사용자에게 공유하고 피드백 수집. 디자인 수정 사항을 디자이너에게 전달.
- **💻 Developer**: 썸네일 디자인 수정 사항에 따라 썸네일 제작 자동화 시스템 업데이트 및 API 연동 확인. 썸네일 생성 프로세스 자동화 검증.
